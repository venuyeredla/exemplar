# Popular Java profilers are :


Profiling is the process of examining an application to identify memory or performance related issues. Java profilers helps in analyzing object creations, method or recursive executions, garbage collectors etc..
	1. JProfiler 
	2. YourKit
	3. Java VisualVM
	4. Netbeans profiler
	5. IntelliJ Profiler
	
## Testing

	4.  Smoke testing refers to preliminary testing. 

	4. Software performance testing.
